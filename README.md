## 📖 Project Overview

This is a command-line Calculator application developed using **Python**.

It performs basic arithmetic operations such as:

- Addition
- Subtraction
- Multiplication
- Division

The project also includes input validation and division-by-zero handling.

This project was developed as part of my **CodSoft Python Internship**.
